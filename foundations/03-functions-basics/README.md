# Functions Basics

Assignment from [The Odin Project - Function Basics](https://www.theodinproject.com/lessons/foundations-function-basics)

## Tasks

- [ ] Write `add7` function - takes one number and returns number + 7
- [ ] Write `multiply` function - takes 2 numbers and returns their product  
- [ ] Write `capitalize` function - capitalizes first letter of string
- [ ] Write `lastLetter` function - returns last letter of string

## Key Concepts

- Function declarations vs expressions
- Parameters vs arguments
- Return values
- Function scope
- Arrow functions

## Run

Open `index.html` in browser and check console.
